    This is a very timing intensive strat that can reward you with 14 seconds of timesave at the start of a run.

    Intermediate - Vault Skippermalink


    Strategy that uses the radio to get out of the vault well before the portal opens. This strategy should be paired with a BSG in Chamber 01 and uses a save load timing that is similar to BSG. This saves a significant amount of time, over 50 seconds if done well.

    Inbounds No SLApermalink

    Beginner - Intendedpermalink

    Advanced - Vault Skippermalink


    Unrestricted Only: Strategy that uses the radio to get out of the vault well before the portal opens. This strategy is similar to the Inbounds Vault Skip however it is done without the save loads like here and is VERY inconsistent. Only recommended when going for top times. This can save a significant amount of time, over 50 seconds if done well.

    Out of Boundspermalink

    Intermediate - Vault Skippermalink


    Strategy that uses the radio to get out of the vault well before the portal opens. This strategy should be paired with a BSG in Chamber 01 and uses a save load timing that is similar to BSG. This strategy can save ~25 seconds over Rubby Save Glitch on version 5135 and is 1-2 seconds faster than performing 02 Skip on version 3420.

    Advanced - Rubby Save Glitchpermalink

    A way to lose collision after the first portal opens, letting you fly right to the elevator at the end. (alternate launch setup)

    Advanced - Omelette Skippermalink

    Build 3420 Only
    The fastest route when paired with Vault Skip. Uses the cube to get save glitch to skip the 02 dialogue. Refer to the old 02 skip video for getting on top of the elevator and getting into the room.

    Expert - 02 Skippermalink

    Build 3420 Only
    This is the most reset heavy trick in the run and recommended to not do until at least sub 8:00. To get the launch more consistently: Jump after the radio touches the wall, and before Chell touches the radio.
